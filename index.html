Optimizing Bus Scheduling for Atal Indore City Transport System (AICTSL)

This project optimized bus scheduling for Indore’s BRTS by analyzing passenger demand and using simulations in R Studio to determine the ideal number of buses for peak and non-peak hours. The findings help improve efficiency, reduce wait times, and enhance public transport reliability.

Market Analysis of Airbnb Listings in Paris

This project is a market analysis of Airbnb listings in Batignolles, Paris, conducted using R to explore factors influencing pricing, availability, and guest satisfaction.
